🛒 E-Commerce MERN Stack Project
📌 Project Overview
This is a full-stack E-Commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The project demonstrates real-world e-commerce functionalities including authentication, product management, cart system, secure payments, and product reviews. It is designed to provide a smooth shopping experience for users and a powerful admin interface for managing the platform.
🚀 Features
👤 User Features
User registration and login with JWT authentication
Browse and search products
View detailed product information
Add and remove products from cart
Place orders and view order history
Product reviews and ratings system
💳 Payment Features
Secure payment integration using Razorpay / Stripe
Automatic order confirmation after successful payment
Payment status tracking
🔐 Authentication & Security
JWT-based authentication for secure login sessions
Password hashing using bcrypt
Protected routes for users and admin access control
Role-based authorization system
